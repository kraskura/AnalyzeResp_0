Description

A function that converts the oxygen level measurements to a different unit as provided. A modified function from: rMR::DO.unit.convert

Updates to increase flexibility in progress

Arguments

Argument Description Default
csv.data The name of the data file where the units need to be converted. A character. One file must be provided. –
units_from The units of the recorded O2 levels. pct = percent air saturation, default. "pct"
units_to The units that the O2 levels will be converted to. mg L^-1 is default "mg/L"
channels Specifies the channels for which the conversion is necessary c(1,2,3,4)
salinity Salinity of the water, in ppt. –
atm_pressure Atmospheric pressure. in Atm; default = 1 1
n_ch The number of channels of the FireSting. If 2-Channel FireSting was used, this argument could be ignored. 4

Value


Details


  Data outputs

“.csv files”

  • New files with converted O2 units
  • File is saved in locally created directory: “./csv_files_unitConvert/”
    • If the directory does not exist, it will be created with a message in the console: "Creating a new directory to store converted files: ./csv_files_unitConvert/"
    • If the directory exists, the new files will automatically be placed in this wd.

  Messages on Console

"Creating new directory to store converted files: ./csv_files_unitConvert/"


Example


The online provided example code has a loop that is handy when transforming all files in a given working directory

 

# For a single file example: 
convert.o2.Firesting(csv.data = "FileToCovert.csv", n_ch = 4, sal = 35)
